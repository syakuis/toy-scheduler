# Scheduler - Toy Project

서버와 클라이언트가 실시간으로 데이터를 상호작용할 수 있도록 구현한다.

### 개발언어

- Spring boot
- Quartz Scheduler
- Ehcache
- Nashorn
- Reactjs
- Mobx
- axios
- Socket.io
